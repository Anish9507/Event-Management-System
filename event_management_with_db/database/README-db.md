
# Database Setup Guide

## MySQL
1. Open MySQL terminal or phpMyAdmin.
2. Run the following command:
   ```sql
   SOURCE database/schema_mysql.sql;
   ```

## PostgreSQL
1. Open PostgreSQL terminal (psql).
2. Run the following command:
   ```sql
   \i database/schema_postgres.sql;
   ```

This will create the required tables for the Event Management Project.
